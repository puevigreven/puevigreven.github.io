Hi, I'm Pralay.

I am a Platform Engineer working at a healthcare startup, ChironX. I primarily work with medical images to detect multiple clinical features using Deep Learning techniques.

I work on various stages of deep learning project, from the implementation of a data lake, developing image annotation software to computer vision tasks like Image Classification, Object Localization, Semantic Segmentation etc. and model deployments.

Before joining ChironX, I did my undergraduate at IIT Patna, with my thesis work supervised by Dr Pushpak Bhattacharyya. As a part of my thesis, I worked closely with Dr Shweta Yadav on Patient Severity Detection task using Natural Language Processing and developed text annotation tools that are still been used in AI-NLP-ML Lab, IIT Patna.
